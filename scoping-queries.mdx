## Why scope queries?

By default, Ravell searches across all connected sources (Linear, Notion, Intercom, Slack). Sometimes you want to narrow the search:

- **Faster answers**: Fewer sources mean less data to search.
- **Focused results**: When you know the answer is in one tool (e.g. Linear only).
- **Time bounds**: Limit to recent data (e.g. "last 7 days") for more relevant results.

---

## Scope to specific sources

You can limit which sources Ravell searches using either the UI or inline syntax:

### Source selector

Click the **source icon** next to the query input to open the source picker. It shows your connected sources (Linear, Notion, Intercom, Slack, Attio) as toggles — switch off the sources you don't want to search. When no sources are selected, all connected sources are queried.

### @mention syntax

Type `@` followed by a source name directly in your question:

- `@Linear What are the top open issues?` → searches only Linear.
- `@Intercom What did customers say about pricing?` → searches only Intercom.
- `@Slack @Linear What did the team discuss about the API?` → searches both Slack and Linear.

Source mentions are case-insensitive. Supported names: `@Linear`, `@Notion`, `@Intercom`, `@Slack`, `@Attio`.

---

## Scope to time ranges

You can scope by time using natural language in your question or the filter panel.

### Natural language

- "What did we ship **last week**?"
- "Summarize Intercom feedback from **last month**."
- "What are the top issues **in our current cycle**?"

Ravell interprets relative time ranges and scopes the search accordingly. For cycle-based questions, Linear cycle dates are used when Linear is connected.

### Filter panel

Click the **filter icon** next to the query input and select a time period:

- **This Week** — current calendar week.
- **This Month** — current calendar month.
- **This Quarter** — current calendar quarter.
- **All Time** — no time restriction.

---

## Filter by priority and team

The filter panel also supports:

- **Priority** — filter results to Urgent, High, Medium, or Low priority items. Select one or more priority levels.
- **Linear teams** — when Linear is connected, filter results to specific teams.

Active filters appear as colored chips below the query input. Click the X on a chip to remove it, or **Clear all** to reset all filters.

---

## Scoping in Agents

When creating an Agent, you can set a default scope:

- **Sources**: Limit the agent to specific sources (e.g. Linear only).
- **Time**: The question text can include time bounds (e.g. "this week", "current cycle").

This keeps agent reports focused and consistent.
