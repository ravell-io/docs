## Global shortcuts

| Shortcut | Action |
|----------|--------|
| **Cmd+N** (Mac) / **Ctrl+N** (Windows) | New Ask |
| **Cmd+K** (Mac) / **Ctrl+K** (Windows) | Open chat search (command palette) |
| **ESC** | Close dialogs |

---

## Chat search (command palette)

Press **Cmd+K** (or **Ctrl+K**) to open the chat search. Type to find past Asks by title, then select one to open it. Useful when you have many conversations.

---

## Workspace switcher

When you belong to multiple workspaces, use **⇧1** through **⇧9** to switch to the first nine workspaces. The workspace picker in the sidebar shows these shortcuts.

---

## Deep links

You can link directly to specific views or Asks:

| URL | Opens |
|-----|-------|
| `https://app.ravell.io/app?view=inbox` | Inbox |
| `https://app.ravell.io/app?view=automations` | Automations |
| `https://app.ravell.io/app?view=sources` | Sources |
| `https://app.ravell.io/app?view=context` | Context |
| `https://app.ravell.io/app?chat=<slug>` | Specific Ask by slug |

Use these in docs, Slack, or email to send teammates to the right place.
